Swapped Tax ID and NPI

If we identify a claim payment that has the NPI and Tax ID numbers swapped, please submit an email to TLCrimsonCSRs@crimsoncsr.com and ClaimsCSManagement@blackhawktpa.com so that we can request the correction.
Make sure you specify "NPI/TAX ID Swapped" in the subject line. Corrections like this do not require a W9, but if you label it as an update request, it may delay the completion of the request!
