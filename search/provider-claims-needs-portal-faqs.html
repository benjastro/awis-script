PROVIDER CLAIMS/NEEDS PORTAL FAQs

  • What is needed for a provider to register?
    o The federal tax id number and company email address

  • What will the username be for provider first time registration?
    o The federal tax id number

  • What will be the password be for provider first time registration?
    o The federal tax id number

  • What does it mean if a provider gets an error message during first registration using their federal tax id#?
    o If a provider gets an error message during first time registration using their federal tax id number, this means the federal tax id number and the provider are not registered in our payment database. To correct this the provider will need to contact Blackhawk Claims Service and submit a W-9 for verification.

  • Can a provider see plan benefits ex: copays and plan details?
    o No, there are no plan details shown in the portal. Providers can only see claims submission and eligibility
